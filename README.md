## ⚒️ Features

- **Play vs friend👦**

- **Play vs computer - computer choose by random💻**

- **Updating score table💯**

- **Choose your own size game board(up to 8x8)🕹️**

- **For chemicales/ not chemicales gas trucks**

- **Run it and describe many cool options**

4. **Enjoy!**
